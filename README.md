# UrFinance
<img src="/image/5.jpg" width="1000"  height="300" style="max-width:100%;">

### Abtract
แอพนี้จัดทำขึ้นเพื่อลดการเสียเวลาในการทำรายรับรายจ่ายในกระดาษ เพื่อที่จะสะดวกต่อการใช้ในชีวิตประจำวันได้ และง่ายต่อการเข้าถึงเพียงแค่ใช้แอพเป็นตัวกลางในการทำรายการ รายรับและรายจ่ายของคุณในแต่ละเดือน เพื่อที่จะดูปริมาณในการใช้เงินแต่ละเดือนอีกทั้งยังเป็นตัวเลขให้เราเห็นอย่างชัดเจน เพื่อเตือนตัวเองว่าควรใช้จ่ายเงินอย่างไรถึงจะสมควรแก่รายรับแต่ละเดือนของเรา และยังสรุปผลให้ว่าเราเหลือเก็บในแต่ละเดือน หรือ เหลือเก็บตลอดทั้งปีเท่าไหร่

## How To Use
<p>เปิด Terminal ขึ้นมาแล้วเข้าไปที่โฟลเดอร์ของโปรแกรมแล้วพิม ./URFinace.out เพื่อเป็นการรันโปรแกรม<p>
  <img src="/image/4.jpg" width="900"  height="300" style="max-width:100%;">
<p> เลือกหัวข้อต่างๆ โดยการพิมตัวเลขตามหัวข้อที่กำหนด เช่นถ้าเลือกหัวข้อแรกจะเป็นการเลือกเดือนเพื่อใส่ รายรับหรือรายจ่าย <p>
  <img src="/image/2.jpg" width="900"  height="300" style="max-width:100%;"><br>
  <img src="/image/3.jpg" width="900"  height="300" style="max-width:100%;">
<p> เมื่อเสร็จแล้วจะกลับมาหน้าแรกอีกรอบเพื่อให้เลือกหัวข้อที่จะทำต่อไป ซึ่งมีให้เลือก 5 หัวข้อคือ<br>
  1. เลือกเดือน เพื่อไปใส่รายรับหรือรายจ่าย<br>
  2. เลือกดูเงินทั้งหมดจากการคำนวณแล้ว (หักลบรายรับรายจ่ายเรียบร้อย)<br>
  3. เลือกดูปริมาณรายรับทั้งหมด<br>
  4. เลือกดูปริมาณรายจ่ายทั้งหมด<br>
  5. เป็นการออกจากโปรแกรม<br>
